Introduction
============
Twaa Mtaro dashboard is a tool to view Twaa Mtaro Infographics. 
The main repository for the project is at ([Twaa Mtaro](https://github.com/navish/twaa-mtaro)).

Installation
------------

- Fork the repository ([here is the guide](https://help.github.com/articles/fork-a-repo/)).
- Clone to your machine
- Install Angular 2 Client.
```bash
npm install -g @angular/cli
```
- Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/twaamtaro-dashboard.git
```

- Install the packages
```bash
cd twaamtaro-dashboard
npm install
```

Running the application
------------
- On the folder project
```
ng serve
```
- For starter page Navigate to [http://localhost:4200/](http://localhost:4200/)
- For admin page Navigate to [http://localhost:4200/admin](http://localhost:4200/admin)

Browser Support
---------------
- IE 9+
- Firefox (latest)
- Chrome (latest)
- Safari (latest)
- Opera (latest)
